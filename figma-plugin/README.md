# Medialive KP Autofill (Figma Plugin)

Плагин для автозаполнения КП двумя способами:
- **JSON mode** — загрузка по URL/вручную,
- **Form mode** — ввод полей в форме (без JSON-файлов).

Плагин заменяет плейсхолдеры вида `{{key}}` в текстовых слоях.

## 1) Установка (Desktop)

1. Откройте Figma Desktop.
2. `Plugins` -> `Development` -> `Import plugin from manifest...`
3. Укажите `figma-plugin/manifest.json`.

## 2) Подготовка макета

Вставьте в текстовые слои плейсхолдеры, например:
- `{{client}}`
- `{{project_name}}`
- `{{offer_no}}`
- `{{screen_dimensions_mm}}`
- `{{sale_total_rub}}`

## 3) Использование

### A) Form mode (рекомендуется, без JSON)
1. Откройте вкладку **Форма (без JSON)**.
2. Заполните поля.
3. Нажмите **Применить форму**.

### B) JSON mode
1. Откройте вкладку **JSON**.
2. Вставьте URL или JSON вручную.
3. Нажмите применить.

## 4) Что важно

- Если ключа нет в данных — `{{key}}` остаётся как есть.
- Плагин обновляет только текстовые слои, где найдены плейсхолдеры.
- Можно работать по выделению (selection) или по всей странице.
