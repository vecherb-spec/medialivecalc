# Medialive KPI Autofill (Figma Plugin)

Мини-плагин для Figma, который:
- получает JSON по URL (например, из вашего калькулятора),
- ищет текстовые плейсхолдеры вида `{{key}}`,
- подставляет соответствующие значения.

## 1) Установка в Figma (Desktop)

1. Откройте Figma Desktop.
2. `Plugins` -> `Development` -> `Import plugin from manifest...`
3. Укажите файл:
   - `figma-plugin/manifest.json`

После этого плагин появится в `Plugins` -> `Development`.

## 2) Подготовка макета

В текстовых слоях используйте плейсхолдеры:

- `{{project_name}}`
- `{{client}}`
- `{{date}}`
- `{{screen_dimensions_mm}}`
- `{{resolution_px}}`
- `{{sale_total_rub}}`

Можно использовать любые ключи, если они есть в JSON.

## 3) Использование

1. Запустите плагин.
2. Вставьте URL JSON (например, опубликованный endpoint калькулятора).
3. Нажмите **Заполнить весь документ**  
   или **Заполнить выделение**.

Плагин:
- находит все `{{...}}` в текстах,
- загружает нужные шрифты,
- заменяет на значения.

## 4) Формат данных

Ожидается JSON-объект:

```json
{
  "project_name": "КП #66",
  "client": "ООО Ромашка",
  "sale_total_rub": 640744
}
```

Если приходит массив — плагин берет первый объект.

## 5) Замечания

- Для некоторых шрифтов Figma может не дать загрузить font (ограничения/доступность).
- Если ключ отсутствует в JSON — плейсхолдер остается как есть.
- Для Google Drive-ссылок используйте прямой URL на JSON, доступный без авторизации.
